# Registro de Decisões Arquiteturais (ADRs) — EOS Framework

Esta pasta armazena o histórico de decisões que moldam os conceitos, diretrizes e o formato do próprio Engineering Operating System (EOS) como produto de engenharia.

---

## Índice de Decisões

* **[ADR-001 — Criação do EOS e Definições Fundamentais de Design](ADR-001-criacao-eos.md)**
  * **Status**: Aprovada
  * **Resumo**: Registra a justificativa técnica para o EOS ser baseado em arquivos Markdown agnósticos de stack e localizados ao lado do código, ao invés de um portal web dedicado.
* **[ADR-002 — Camada Operacional de Prompts no EOS Framework](ADR-002-camada-operacional-prompts.md)**
  * **Status**: Aprovada
  * **Resumo**: Justifica a criação da pasta `prompts/` contendo instruções executáveis para garantir a repetibilidade e conformidade dos processos por desenvolvedores e IAs.
